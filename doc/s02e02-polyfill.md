# vstack/lib/polyfill

## [vstack/lib/polyfill/es6-promise]

Promises polyfill with ES6 API. Depends on [es6-promise].

### Usage

```js
require('vstack/lib/polyfill/es6-promise');
expect(window.Promise).toBeDefined();
```
